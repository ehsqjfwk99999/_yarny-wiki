Kaggle
======

API
---

### Installation
```sh
    pip install kaggle
```

### API token
- Create API token(kaggle.json) from [here](https://www.kaggle.com/grumyo99999/account?isEditing=False).
- Linux Location.
```
    ~/.kaggle/kaggle.json
```
-  Change API token authority.
```sh
    chmod 600 ~/.kaggle/kaggle.json
```
