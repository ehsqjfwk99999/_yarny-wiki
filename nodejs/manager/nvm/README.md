nvm
---

### - Show manual page.
```sh
nvm --help
```
### - Show version.
```sh
nvm --version
```

nvm ls
------

### - List installed versions.
```sh
nvm ls
```

nvm install
-----------

### - Install the latest available version.
```sh
nvm install node
```
### - Install the latest LTS version.
```sh
nvm install --lts
```

nvm use
-------

### - Use the latest version.
```sh
nvm use node
```
### - Use the latest LTS version.
```sh
nvm use --lts
```
