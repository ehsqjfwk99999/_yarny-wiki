nvm
---

```sh
# Show manual page.
nvm --help

# Show version.
nvm --version
```

nvm ls
------

```sh
# List installed node versions.
nvm ls
```

nvm install
-----------

```sh
# Install the latest available node version.
nvm install node

# Install the latest node LTS version.
nvm install --lts
```

nvm use
-------

```sh
# Use the latest node version.
nvm use node

# Use the latest node LTS version.
nvm use --lts
```
